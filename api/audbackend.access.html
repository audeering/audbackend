access()¶

audbackend.access(name, host, repository)[source]¶

Access repository.

Returns a backend instance for the repository on the host. The instance is an object of the class registered under the alias name with audbackend.register().

If the repository does not exist or cannot be accessed for other reasons (e.g. user lacks permission) an error of type audbackend.BackendError is raised.

Use audbackend.available() to list available backend instances.

Parameters

  • name (str) – alias under which backend class is registered

  • host (str) – host address

  • repository (str) – repository name

Return type

Backend

Returns

backend object

Raises

  • BackendError – if an error is raised on the backend, e.g. repository does not exist

  • ValueError – if no backend class with alias name has been registered

Examples

>>> access('file-system', 'host', 'doctest')
('audbackend.core.filesystem.FileSystem', 'host', 'doctest')
